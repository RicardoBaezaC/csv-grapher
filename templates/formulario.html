About

Esta página esta realizada con la finalidad de optimizar el tiempo en el que se realiza una representación gráfica de datos, la página consiste en que el usuario debe subir un archivo con extención csv y seleccionar tipo de grafica que desea y la página realizara la tarea en segundos. A demás de que la página presenta una interfaz gráfica amigable y fácil de utilizar. Este pequeño proyecto esta realizado con el lenguaje de programación Python y las librelias Flask, Pandas y Matplotlib.



Herramientas


Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código.2​ Se trata de un lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma.
Flask es un framework minimalista escrito en Python que permite crear aplicaciones web rápidamente y con un mínimo número de líneas de código. Está basado en la especificación WSGI de Werkzeug y el motor de templates Jinja2 y tiene una licencia BSD.
Matplotlib es una biblioteca para la generación de gráficos a partir de datos contenidos en listas o arrays en el lenguaje de programación Python y su extensión matemática NumPy. Proporciona una API, pylab, diseñada para recordar a la de MATLAB.
Pandas es una herramienta de análisis y manipulación de datos de código abierto rápida, potente, flexible y fácil de usar, construida sobre el lenguaje de programación Python.
